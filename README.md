# Eleva

An AI-powered placement preparation platform that helps students prepare for technical and behavioral interviews through personalized mock interviews, resume-aware questioning, AI-driven feedback, and long-term performance analytics.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL (Planned)
- Prisma (Planned)
- OpenAI API (Planned)
